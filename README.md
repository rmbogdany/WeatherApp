# Weather App

Cordova Application which displays weather at selected location using open source APIs.

## API Usage

  - Leaflet Map Api
  - OpenWeatherApi
  - Nominatim's OpenStreetMap Api

## Cordova Plugins
   
   - Cordova Geolocation plugin
